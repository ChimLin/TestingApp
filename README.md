# TestingApp

# Quick start
git clone https://github.com/RuslanKhaliullin/TestingApp.git <br />
cd demo2 <br />
mvn clean install  <br />
java -jar target/demo-0.0.1-SNAPSHOT.jar path to csv file path to json file <br />
